# Deehamlin.github.io

## Installation
- clone the repository to work on your machine
- install ruby
- install jekyll

## Running Locally
- cd into the directory where the repo was cloned
- run "jekyll build"
- run "jekyll serve"
- view page at http://localhost:4000/

## Adding new posts
- create a new branch
- add post in "\_posts" directory
